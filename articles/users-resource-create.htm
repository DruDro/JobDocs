Users
Create A New User Call

Adds a new User object in the system

URL

api/Users

Method

POST

Params

  • user - UserRegistrationDTO

Success Response

  • Code: 
    200 string
  • Content: 
    The created Serialized UserRegistrationDTO

Sample cURL

curl -X POST --header 'Content-Type: application/json' --header 'Accept: application/json' -d 'UserRegistrationDTO' 'http://10.0.2.18:4001/api/Users'

Sample Value 

{
  "FullName": "string",
  "Email": "string",
  "Password": "string",
  "ZipCode": "string",
  "FacebookId": "string",
  "GoogleId": "string",
  "ProfilePicBlobUri": "string"
}

Model 

UserRegistrationDTO {
  FullName (string, optional),
  Email (string, optional),
  Password (string, optional),
  ZipCode (string, optional),
  FacebookId (string, optional),
  GoogleId (string, optional),
  ProfilePicBlobUri (string, optional)
}
