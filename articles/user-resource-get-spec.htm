User Resource
Get A Specific User Call

returns a specific User object in the system

URL

api/v1/Users/{id}

Method

GET

Params

  • id - the id of the User to retrieve.

Payload

None

Required Security

Authenticated, Role based restrictions

Success Response

  • Code: 
    200
  • Content: 
    Serialized UserDTO

Sample Code

curl -X GET --header 'Accept: application/json'
'https://jdmobileapp.azurewebsites.net/api/v1/Users/7c3a729f-b478-4b90-bafb-dd29493e33e9'

Notes

While it may suit the mobile app to request a job and get the entire hierarchy it needs to handle user registration, if it causes problems it can be paired down and broken out into seperate calls.
