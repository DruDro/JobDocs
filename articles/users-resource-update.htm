Users
Update A Specific User Call

Updates a specific ApplicationUser object in the system

URL

api/Users/{id}

Method

PUT

Params

  • id - user id to update
  • user - UserDTO

Success Response

  • Code: 
    200 OK
  • Content: 
    The updated Serialized ApplicationUserDTO

Sample cURL

curl -X PUT --header 'Content-Type: application/json' --header 'Accept: application/json' -d 'UserDTO' 'http://10.0.2.18:4001/api/Users/{id}'
