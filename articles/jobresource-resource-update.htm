JobResource Resource
Update A Specific JobResource Call

Updates a specific JobResource object in the system

URL

api/v1/JobResources/{id}

Method

PUT

Params

  • id - the id of the JobResource to update.

Payload

serialized JobResourceDTO

Required Security

Authenticated, Admin role ONLY!

Success Response

  • Code: 
    200
  • Content: 
    The updated Serialized JobResourceDTO

Sample Code

curl -X PUT --header 'Accept: application/json'
