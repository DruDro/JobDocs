User Flow Description

This section assumes that you have already read the following articles:

  • Sign Up / Log In
  • Home Screen
  • Working With Documents

There are several preselected categories in JobDocs, based on your job:

  • Licenses - a predefined category, designed to store licenses.

  • Vaccine Complience - a predefined category, designed to store vaccine complience documents.

  • Skill Certifications - a predefined category, designed to store certifications.

  • Contact Hours (CE) - a predefined category, designed to store Contact Hours certificates. On the Home Screen, it displays not the number of documents, but the earned time of Continued Education, Contact Hours, Clinical Hours, Rx Hours, depending on what you specify when adding a document.

  • Diplomas - a predefined category, designed to store diplomas.

  • Job-Related Equipment - a predefined category, designed to store information about equipment you can use.

  • Receipts - a predefined category, designed to store receipts.

  • Contracts - a predefined category, designed to store contracts.

  • Documents - a predefined category, designed to store other kinds of documents.

When creating an account in JobDocs, you can see the list of available categories.

The selected categories will be displayed on the Home Screen. From here you can add, edit, share and remove categories.

Adding a category

To add a new category, follow the steps below:

  • Tap the button

  • Select Category

  • Type in the name of the new category.

  • CREATE

The newly created category will appear in the Categories list on the Home Screen. So now you can add or move documents to this category.

Create a new category while adding documents

If you need to add a new document to a category, that doesn't exist yet, you can skip the above steps. Just perform the following steps:

  • Tap the button

  • Select Document

  • Follow the steps described in the Working With Documents article, the Adding a new document section.

  • When you reach the step where the new document page appears:

    tap Set Location -> Create Category

  • Type in the name of the new category.

  • Create

  • Follow the instructions from the Working With Documents article.

Renaming a category

On the Home Screen, you can see the list of your document categories. You can edit category name by calling the category shortcut menu. For this:

  • tap the ... button next to the needed category name

    or tap the needed category in the list - the category page appears. Tap the ... button in the header.

  • Select Rename

  • Type in the new name

  • Yes

You can't rename the predefined categories.

Removing a category

To remove a category from the list in your profile, call the category shortcut menu as described in the Renaming a category section, select Delete, confirm.

Sharing categories

You can share categories from JobDocs to any cloud, e-mail, messenger, social network, etc., that is configured on your device. From the category shortcut menu, select share, tap the needed sharing service and share the category like you always share a file.

For more details, see the Sharing article.

Category toolbar

On the category page, here is a toolbar with the following capabilities:

  • List / Tiles View switcher - /

  • Sort by dropdown menu to sort items in a category:

    • by Issued date

    • by Expiration date

    • or in Alphabetical order
