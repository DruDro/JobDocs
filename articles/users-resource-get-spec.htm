Users
Get A Specific User Call

Returns a specific User object in the system

URL

api/Users/{id}

Method

GET

Params

  • id - id of a specific User to get

Success Response

  • Code: 
    200 OK
  • Content: 
    The Serialized ApplicationUserDTO

Sample cURL

curl -X GET --header 'Accept: application/json' 'http://10.0.2.18:4001/api/Users/{id}'
