Category Resource
Update A Specific Category Call

Updates a specific Category object in the system

URL

api/v1/Categories/{id}

Method

PUT

Params

  • id - the id of the Category to update.

Payload

serialized CategoryDTO

Required Security

Authenticated, Role based restrictions

Success Response

  • Code: 
    200
  • Content: 
    The updated Serialized JobDTO

Sample Code

curl -X PUT --header 'Accept: application/json'
'https://jdmobileapp.azurewebsites.net/api/v1/Categories/e5d8efbf-5bc9-4535-b824-7602689c18b5'

Notes

Admin role can update any type of category while a user role can only update a non-system category owned by the authenticated user
