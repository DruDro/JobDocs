Job Resource
Get All Jobs Call

returns all Job objects in the system

URL

api/v1/Jobs

Method

GET

Params

None

Payload

None

Required Security

Anonymous

Success Response

  • Code: 
    200
  • Content: 
    Serialized JobDTO, with contained JobRoleDTOs and JobResourceDTOs

Sample Code

curl -X GET --header 'Accept: application/json' 'https://jdmobileapp.azurewebsites.net/api/v1/Jobs'

Notes

While it may suit the mobile app to request all jobs and get the entire hierarchy it needs to handle user registration, if it causes problems it can be paired down and broken out into seperate calls.
