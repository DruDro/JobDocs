Auth
Confirm Email Call

URL

api/Auth/ConfirmEmail?token={token}&id={id}

Method

GET

Params

  • token - email confirmation token
  • id - user id

Success Response

  • Code: 
    200 OK
  • Content: 
    None

Sample cURL

curl -X GET --header 'Accept: application/json' 'http://10.0.2.18:4001/api/Auth/ConfirmEmail?token={token}&id={id}'
