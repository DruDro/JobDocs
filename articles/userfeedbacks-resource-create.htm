UserFeedback Resource
Create A New UserFeedback Call

Adds a new UserFeedback object in the system

URL

api/v1/UserFeedback/{id}

Method

POST

Params

  • id - the id of the UserFeedback to create.

Payload

serialized UserFeedbackDTO

Required Security

Authenticated

Success Response

  • Code: 
    201
  • Content: 
    The created Serialized UserFeedbackDTO

Sample Code

curl -X POST --header 'Accept: application/json'
'https://jdmobileapp.azurewebsites.net/api/v1/UserFeedbacks/e5d8efbf-5bc9-4535-b824-7602689c18b5'

Notes

Authenticated users can only post feedback associated with themselves.
