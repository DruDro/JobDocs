User Flow Description

Adding a new document

To add a new document to JobDocs, follow the steps below:

  • Tap the button on the bottom toolbar of the application.

  • Select whether to use a photo from the library on your device or to take a new photo.

    JobDocs needs access to the file system and camera on your device. Tap OK when you are aksed to permit access. Or open the device Settings -> JobDocs -> setup access to device Camera and Photos. All your data will remain confidential.

  • If you select to take a photo, use the camera and take one or more photos of a document.

    If you select to add a photo from the library, select one or more photos and tap Ready.

  • You can adjust each photo (crop and rotate) at this step or later - in the file settings. To adjust a photo, tap the button.

  • When ready, tap the button.

  • The document page appears.

    All the photos you have taken at the previous steps, are stored and related to a document file. You can look through all the photos by swiping them.

  • Next you should identify the document:

    • Give it a File name:
      Tap on the placeholder to edit the file name.

    • Set location - select which category will store the document:
      Tap on the Set location placeholder -> tap a category you need. The category will expand and display all the documents stored in it (if there are any).

      For more details about categories, see the Categories topic.

      When ready, tap Set Location.

    • Set expiration date.

    • Add notes.

  • Tap SAVE.

The document is configured and saved:

Now you can edit or share it if needed.

Editing a document

The following instructions assume that you already have expirience with Adding documents.

To edit an existing document, you need to call the shortcut menu including the Edit shortcut.

This shortcut menu can be called in one of the following ways:

  1. From the Search page:

  2. From a Category page:

  3. On the document page:

Tap Edit. You will get to the document editing mode.

Here you can:

  • Add images.
  • Change file name.
  • Set file location - change the category to contain the document. For more details about categories, see the Categories article.
  • Change expiration date.
  • Edit notes.

When done, tap SAVE. Your edits will be saved.
