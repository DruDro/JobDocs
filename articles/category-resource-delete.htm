Category Resource
Delete A Category Call

Deletes a Category object in the system

URL

api/v1/Categories/{id}

Method

DELETE

Params

  • id - the id of the Category to delete.

Payload

None

Required Security

Authenticated, Admin role ONLY!

Success Response

  • Code: 
    204
  • Content: 
    none

Sample Code

curl -X DELETE --header 'Accept: application/json'
'https://jdmobileapp.azurewebsites.net/api/v1/Categories/e5d8efbf-5bc9-4535-b824-7602689c18b5'

Notes

Admin role can delete any type of category (within referential constraints) while a user role can only delete a non-system category owned by the authenticated user.
