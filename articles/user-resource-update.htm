User Resource
Update A Specific User Call

Updates a specific User object in the system

URL

api/v1/Users/{id}

Method

PUT

Params

  • id - the id of the User to update.

Payload

multipart - serialized UserDTO and profile photo image (image only if it is being updated)

Required Security

Authenticated, Role based restrictions

Success Response

  • Code: 
    200
  • Content: 
    The updated Serialized UserDTO

Sample Code

curl -X PUT --header 'Accept: application/json'
'https://jdmobileapp.azurewebsites.net/api/v1/Users/e5d8efbf-5bc9-4535-b824-7602689c18b5'

Notes

Admin role can update any user while a user role can only update themselves. Certain items may not be updated at this time, such as their Job. THre may be other items that don't make sense to let them modify.
