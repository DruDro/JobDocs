Category Resource
Create A New Category Call

Adds a new Category object in the system

URL

api/v1/Categories/{id}

Method

POST

Params

  • id - the id of the Job to create.

Payload

serialized CategoryDTO

Required Security

Authenticated, Role based restrictions

Success Response

  • Code: 
    201
  • Content: 
    The created Serialized CategoryDTO

Sample Code

curl -X POST --header 'Accept: application/json'
'https://jdmobileapp.azurewebsites.net/api/v1/Categories/e5d8efbf-5bc9-4535-b824-7602689c18b5'

Notes

Admin role can create any type of category while a user role can only create a non-system category that will be owned by the authenticated user.
