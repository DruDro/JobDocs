User Resource
Create A New User Call

Adds a new User object in the system

URL

api/v1/Users/{id}

Method

POST

Params

  • id - the id of the User to create.

Payload

multipart - serialized UserDTO and profile photo image

Required Security

Authenticated, Admin role ONLY!

Success Response

  • Code: 
    201
  • Content: 
    The created Serialized UserDTO

Sample Code

curl -X POST --header 'Accept: application/json'
