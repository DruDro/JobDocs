User Flow Description

Notifications page

To see the upcoming document expiration notifications, tap the button, that display the number of alerts.

Here you can tap a document to go to its page and, for example, edit it.

Document expiration date

The date specified in the document Edit Mode is then used to notify you when the document expires. Don't forget to set expiration date when adding a document.

Notification settings

You can change when you want to be notified. For this, tap the button. Here you can:

  • Select a time interval to notify before documents expire.

  • Snooze — select a time interval to repeat notification.
